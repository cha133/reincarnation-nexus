# Reincarnation Nexus

一个为 SillyTavern 角色扮演场景编写的共享世界观项目。

本仓库以“轮回宇宙”为核心，记录系统、宿主、节点、副本、本源与轮回点等共同规则。世界设定可以作为多个系统角色卡的顶层背景，使不同系统与角色在同一套底层规则下展开故事。

## 仓库内容

- [`worldbook/reincarnation-universe.md`](worldbook/reincarnation-universe.md)：轮回宇宙的核心世界设定。
- `system-cards/`：预留用于存放不同系统的角色卡或设定。
- `assets/`：预留用于存放图片等配套资源。

## 使用说明

核心世界设定适合作为角色定义之前的共享上下文。各系统卡可以另外定义自身的能力、性格、阵营、任务模式、商城规则和绑定契约，但未明确声明的部分默认遵循核心世界设定。
